## Modifications

This fork adds the following on top of upstream:

- Dark mode (with View Transitions persistence, dynamic logo switching)
- CSS variable mapping fixes for dark mode
- Bug fixes (logo sizing, hero image aspect ratios, author page covers, similar posts links)
- Typography refinements (Inter default, optional CJK setup)
- Simplified categories (`blog` / `lifestyle`) with default cover images

Modifications copyright (c) 2026, chialin.
